5. The 90-day program

The program is structured across three phases: Break (Days 1–30), Build (Days 31–60), and Control (Days 61–90). Each phase has specific blocking behavior and interaction patterns.

Crisis overrides

Each phase provides a limited number of crisis overrides — temporary deactivations of blocking for genuine emergencies. Overrides are time-limited and logged. The number of remaining overrides per phase is displayed in the app. Use of an override notifies your accountability partner if one is set up.

Phase transitions

Phase transitions occur automatically at Days 31 and 61. Features and restrictions change at each transition as described in the app. There is no refund or credit for early exit from the program.
